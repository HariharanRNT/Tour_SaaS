Plan Trip Page — Full UI Suggestions (Theme-Adaptive)
Since the agent can switch themes, every suggestion below is written as theme-neutral — meaning it works beautifully regardless of whether the active theme is Coral Blaze, Ocean Deep, Sunset Peach, or any other.
All color references use --primary (agent's chosen theme color) instead of hardcoded values.

1. NAVBAR
Priority: Visual Appeal first
Structure suggestion:

Keep the white frosted pill navbar — it works across ALL themes because it is neutral white
Make it background: rgba(255,255,255,0.85) with backdrop-filter: blur(20px) — this way the hero image bleeds through slightly giving it depth
Add border: 1px solid rgba(255,255,255,0.9) and box-shadow: 0 4px 24px rgba(0,0,0,0.08) for a lifted premium feel

Active nav pill (Plan Trip):

Always use --primary color as the pill background
Text inside always white
Add box-shadow: 0 4px 16px var(--primary-glow) — this glow changes per theme automatically
So Coral Blaze = red glow, Ocean Deep = blue glow, Forest Trail = green glow — premium feel across all themes

Inactive nav links:

Color should be a fixed dark warm charcoal #2C2C2C — never inherit --primary for text links, it reduces readability
On hover: text color shifts to --primary with a smooth transition: color 0.2s

Logo:

Globe icon color should always match --primary — this is the one place the logo picks up the agent's theme color
"RNT Travel" text stays dark charcoal always

Right side (logged in state):

Username "Hariharan" in muted charcoal
Logout button: border: 1.5px solid --primary, text in --primary, on hover fills with --primary and text goes white
This makes logout button always theme-consistent without being too aggressive

Scroll behavior:

On scroll down: navbar transitions from fully transparent to rgba(255,255,255,0.95) with a smooth 0.3s transition
This is called a scroll-aware sticky navbar — works beautifully on all themes


2. HERO + SEARCH BAR
Priority: Visual Appeal first
Hero background:

Keep the desert van background photo — the warm terracotta tones complement almost every theme color
Add a dual-layer gradient overlay:

Layer 1: linear-gradient(to bottom, rgba(0,0,0,0.2) 0%, rgba(0,0,0,0.5) 100%) — dark overlay for text readability
Layer 2: a very subtle tint using --primary at 10% opacity — rgba(var(--primary-rgb), 0.10) — this makes the hero image subtly breathe in the agent's theme color


So Coral Blaze hero has a faint warm red cast, Ocean Deep has a faint blue cast — immersive and premium

Hero heading "Where do you want to go?":

Keep the mixed serif italic typography — it is a strong design choice
Increase size to 68–72px on desktop
Add text-shadow: 0 2px 20px rgba(0,0,0,0.4) for depth and readability
The italic want word — add a very subtle color: var(--primary-light) tint to it so it picks up the theme color slightly — elegant touch

Subheading:

"Search for a destination, package name, or an experience you love"
Keep white, reduce opacity to 80%, add same text shadow
Max-width 600px, centered

Search bar:

Container: background: rgba(255,255,255,0.92), backdrop-filter: blur(20px), border-radius: 20px, box-shadow: 0 8px 40px rgba(0,0,0,0.15), border: 1px solid rgba(255,255,255,0.95)
Search icon on left: always in --primary color — changes per theme automatically
Placeholder text cycles: "Try Chennai…" → "Try Kochi…" → "Try Manali…" with fade animation every 2.5 seconds
"Explore Now" button:

Background: linear-gradient(135deg, --primary, --primary-dark)
box-shadow: 0 6px 24px var(--primary-glow)
On hover: lifts with transform: translateY(-1px), glow deepens
This button is the single most important theme-colored element on the page — make it shine



Category pills (Adventure, Beach, Honeymoon, Family):

Current solid black pills completely ignore the theme — fix this
New style: background: rgba(255,255,255,0.18), backdrop-filter: blur(8px), border: 1px solid rgba(255,255,255,0.35), white text, bold, letter-spacing 1px
On hover: background: var(--primary), border becomes var(--primary), text stays white
Add a small icon before each label:

🏔 Adventure · 🏖 Beach · 💑 Honeymoon · 👨‍👩‍👧 Family


This makes category pills theme-reactive on hover while staying neutral by default

Social proof bar (480+ trips planned this week):

Wrap in a frosted glass pill: background: rgba(255,255,255,0.18), border-radius: 50px, padding 8px 20px
Avatar ring border: white 2px solid
"480+" number: bold, color var(--primary-light) — picks up theme color
Small animated green pulse dot next to it suggesting live activity


3. BROWSE BY TRIP STYLE
Priority: Visual Appeal + Theme Consistency
This is the most important section to fix for theme switching.
Section background:

Do NOT use a flat pink/blue/green color that hardcodes one theme
Instead use: linear-gradient(160deg, rgba(var(--primary-rgb), 0.06) 0%, rgba(var(--primary-rgb), 0.12) 100%)
This means the section background is ALWAYS a very soft tint of the agent's theme color
Coral Blaze → very soft rose tint
Ocean Deep → very soft blue tint
Forest Trail → very soft green tint
Beautiful, zero extra work

Section container:

Keep the rounded rectangle container shape — it is a good design choice
Add border: 1px solid rgba(var(--primary-rgb), 0.15) — subtle themed border
border-radius: 28px, generous padding

Section heading "Browse by Trip Style":

Increase to 34px, semi-bold
Color: deep warm charcoal #1A1A1A — fixed, not theme-dependent
This ensures readability across all theme tints

Carousel arrows (← →):

Background: var(--primary) — already good in your current version
Add box-shadow: 0 4px 16px var(--primary-glow) for the glow effect
Icon inside: always white

Trip Style Cards — the biggest improvement needed:
Each card should have its OWN fixed personality color that never changes with theme — because Adventure is always adventurous, Beach is always oceanic, etc. The theme color only appears in the icon and hover state, not the card background:
Adventure card:

Background: soft warm terracotta tint #FFF4EE
Icon background chip: #FFE8DC
Icon color: --primary (theme color) — so coral = red mountain, ocean = blue mountain
Title: deep charcoal
Subtitle: muted brown
Bottom accent bar on hover: var(--primary) 3px height

Beach card:

Background: soft sky tint #EEF8FF
Icon background chip: #DAEFFE
Icon color: --primary

Honeymoon card:

Background: soft blush #FFF0F5
Icon background chip: #FFE0EC
Icon color: --primary

Family card:

Background: soft mint #F0FAF4
Icon background chip: #D8F3E4
Icon color: --primary

Card hover state (all cards):

transform: translateY(-6px)
box-shadow: 0 12px 32px rgba(var(--primary-rgb), 0.18)
Bottom border: 3px solid var(--primary)
Icon chip background deepens slightly

This approach gives each card its own identity while the theme color ties them all together through the icons and hover states.
